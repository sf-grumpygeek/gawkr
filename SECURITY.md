# Security policy

gawkr processes security-camera footage, so please report vulnerabilities
privately rather than in a public issue.

## Reporting a vulnerability

Please use GitHub's **[private vulnerability reporting](https://github.com/sf-grumpygeek/gawkr/security/advisories/new)**
(the "Report a vulnerability" button under this repository's *Security* tab).
That keeps the report private until a fix is available — please don't open a
public issue for anything exploitable.

Include what you found, how to reproduce it, and the impact. Reports about
leaking footage, snapshots, credentials, or the vision endpoint are highest
priority.

## Scope & expectations

gawkr is a personal open-source project maintained on a best-effort basis, not a
staffed security team. I'll acknowledge and address reports as I'm able, but
can't promise a fixed response time. Thanks for reporting responsibly.
